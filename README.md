# scholarship-automation-project

## Project Overview
This project automates the scholarship application process using no-code tools.  
Students submit their details through a Google Form, and an automatic confirmation email is sent.

## How It Works
1. Student fills the Google Form  
2. Data is stored in Google Sheets  
3. Make.com detects new entry  
4. Gmail sends auto-reply email  

## Tools Used
- Google Forms  
- Google Sheets  
- Make.com  
- Gmail
